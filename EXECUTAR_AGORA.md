# 🚀 EXECUTAR EXPO OFFLINE

## Execute este comando no PowerShell:

```powershell
$env:NODE_TLS_REJECT_UNAUTHORIZED="0"
npx expo start --offline
```

## Ou simplesmente:

```powershell
npx expo start --offline
```

## Depois pressione:
- `w` para **Web** (navegador) ✅ **RECOMENDADO**  
- `a` para **Android**
- `i` para **iOS**

## 🌐 URL do App:
```
http://localhost:19006
```

- App React Native: ✅ Funcionando
